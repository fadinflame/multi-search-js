# MultiSearch

> An extension to find alternatives

## Installation

1. Download the latest release
2. Unpack the archive
3. Go to chrome://extensions/
    - Enable developer mode
    - Press "Load unpacked"
    - Select a folder with the extension
4. Profit!!!
